Haar Cascade Object Detection README


This README provides a step-by-step guide on how to train a custom object detection model using the Cascade Trainer GUI to detect cars and bikes. The process involves dataset preparation, setting up the required tools, utilizing the Cascade Trainer GUI, training the model, and evaluating its performance.


GUI Download Link: https://amin-ahmadi.com/cascade-trainer-gui/


Step 1: Open the Folder ‘Haar cascade’.
Step 2: Open ‘custom_object_detection’ file and set the path of the xml files to be detected (cascade_car and cascade_bike).
Step 3: Run the file named ‘custom_object_detection’.
